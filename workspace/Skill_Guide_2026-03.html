🎯 什么是 Skill？

Skill（技能） 是 OpenClaw 的能力扩展模块，让 AI 助手获得特定领域的专业操作能力。就像手机 APP 一样，一个 skill = 一个功能模块。

💡 为什么需要 Skills？
✅ 开箱即用 - 不用自己开发复杂逻辑
✅ 标准化流程 - 每个 skill 都遵循最佳实践
✅ 扩展性强 - 可从 clawhub.com 下载更多 skills
✅ 场景丰富 - 从文件处理到设备控制一网打尽

🔥 Top 5 强烈推荐 Skills

🥇 #1: coding-agent（编程代理）

"自己写代码太慢？让 Codex/Claude Code/Pi Agent 替你干！"
核心能力：
  • 自动搭建项目脚手架
  • 生成完整功能模块
  • 审查 PR / Debug 报错日志
  • 重构大型遗留代码
安装方式（已预装）： 
# 确认存在
Test-Path "C:\Users\huangxiaolong\AppData\Roaming\npm\node_modules\openclaw\skills\coding-agent"
使用示例：
  • "用 Python 写一个爬虫抓股票数据"
  • "帮我优化这段 SQL 查询效率"
  • "Review 这个 PR，找出潜在 bug"

🥈 #2: nano-pdf（PDF 编辑）

"Word 能改，PDF 为什么不能？用它直接编辑 PDF！"
核心能力：
  • 文字增删/替换/高亮
  • 添加注释与批注
  • 提取文本转 Markdown
  • 合并/拆分 PDF（CLI）
安装方式（需补充依赖）： 
# 检查是否已装 nano-pdf CLI  
nano-pdf --version

# 如果缺失则安装
npm install -g @artificialio/nano-pdf-cli
使用示例：
  • "把合同第 3 页的价格改成¥50,000"
  • "提取这份 PDF 里所有的表格数据"
  • "把这个 PDF 转成 Markdown"

🥉 #3: feishu-doc（飞书文档管理）

"不用切到浏览器，直接命令行操作飞书文档！"
核心能力：
  • 创建/写入/追加文档内容
  • 插入表格、图片、富文本
  • 批量更新多人协作文件
  • 上传附件（合同/报告）
使用示例：
  • "创建一个文档《周报_2026-03-09》并填入这些内容..."
  • "在 docx/xxx 文档末尾追加一段总结"
  • "上传这张截图到 Feishu Docs"

#4: healthcheck（系统安全巡检）

📋 安装方式总览

方法一：命令行安装（推荐）

# 从 clawhub.com 下载
openclaw skill install <skill_name>

方法二：手动安装

$destPath = "C:\Users\您的用户名\AppData\Roaming\npm\node_modules\openclaw\skills"
New-Item -ItemType Directory -Force -Path "$destPath\"
